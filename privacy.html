Применяется ко всем персональным данным пользователей сайта, включая заемщиков (залогодателей) и инвесторов (займодавцев, залогодержателей).

Компания оказывает агентские услуги по подбору инвесторов и заемщиков для организации финансирования под залог недвижимости. Компания не является кредитной организацией, банком или МФО.

2. Персональные данные

Оператор может обрабатывать:

  • ФИО, дата рождения
  • паспортные данные
  • адрес регистрации
  • контактные данные (телефон, email)
  • сведения об объектах недвижимости
  • сведения из документов, включая выписку ЕГРН/ЕГРЛ, паспорта, договора и иные документы
  • сведения о доходах, задолженностях, кредитоспособности, семейном положении, наличии детей
  • сведения из БКИ
  • иные документы, направленные пользователем

3. Цели обработки

  • рассмотрение заявки
  • подбор инвестора/займодавца или заемщика/залогодателя
  • организация финансирования под залог недвижимости
  • заключение и исполнение агентских договоров
  • оценка кредитоспособности (включая данные из БКИ)
  • проверка долгов, семейного положения, наличия детей
  • связь с пользователем
  • соблюдение требований законодательства

4. Способы получения и обработки данных

  • через формы сайта
  • по электронной почте
  • через мессенджеры
  • путем направления документов пользователем

Данные обрабатываются с использованием или без использования средств автоматизации.

Оператор выполняет: сбор, хранение, использование, передачу, удаление.

5. Передача данных инвесторам

Передача данных инвесторам осуществляется только для целей оценки заявки и организации финансирования.

Инвесторы обязаны использовать данные только для оценки заявки и соблюдения конфиденциальности.

Передача другим третьим лицам (IT-подрядчики, хостинг) возможна только для обработки заявки и функционирования сайта.
